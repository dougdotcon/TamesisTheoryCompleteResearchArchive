Spectral Analysis

Douglas H. M. Fulber
Universidade Federal do Rio de Janeiro, Rio de Janeiro, Brazil
(Dated: January 23, 2026)
This module analyzes the spectral properties of operators using Random Matrix Theory (RMT). Key focus: comparing level spacing statistics to GUE (Gaussian Unitary Ensemble) predictions, which are characteristic of quantum chaotic systems.

1. Level Spacing Statistics

The distribution of spacings between consecutive eigenvalues reveals the nature of the system:

System Type Spacing Distribution Variance
Integrable (uncorrelated) Poisson: $P(s) = e^{-s}$ $\sigma^2 = 1$
Chaotic (level repulsion) GUE: $P(s) \sim s^2 e^{-s^2}$ $\sigma^2 \approx 0.286$

1.1 Montgomery-Odlyzko Conjecture

The famous conjecture states that the spacings between Riemann zeros follow GUE statistics. This suggests a deep connection between number theory and quantum chaos.

2. Implementation

The file spectral_analysis.py provides:

  • compute_eigenvalues() — Extract eigenvalues from operator matrix
  • level_spacing_distribution() — Compute normalized spacings
  • gue_comparison() — Test compatibility with GUE

3. Results

For the Berry-Keating operator (Phase 1):

  • Variance ratio: ~0.7 (compatible with GUE range)
  • Interpretation: The discretized operator shows chaotic-like level repulsion

Caveat: GUE-like statistics are necessary but not sufficient for connection to Riemann zeros. Many chaotic systems have GUE statistics without any number-theoretic meaning.
