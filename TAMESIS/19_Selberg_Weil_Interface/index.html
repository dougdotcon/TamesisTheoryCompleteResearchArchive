2. The Parallel Formulas

Selberg Trace Formula: $$\sum_n h(r_n) = \frac{\mu}{4\pi} \int h(r) r \tanh(\pi r) dr + \sum_\gamma \frac{\ell_\gamma}{2\sinh(k\ell_\gamma/2)} g(k\ell_\gamma) + \text{cusp}$$
Weil Explicit Formula: $$\sum_\rho h(\gamma_\rho) = h(0) + h(1) - \sum_p \sum_k \frac{\log p}{p^{k/2}} \hat{h}(k \log p) + \Gamma\text{-terms}$$

Key parallel: Eigenvalues $\leftrightarrow$ Zeros, Geodesics $\leftrightarrow$ Primes

3. Zero Proximity and Fluctuations

The oscillatory term $E_\zeta(T)$ in our scattering phase decomposition is controlled by Riemann zeros near height $2T$:

$$E_\zeta(T) \sim \frac{1}{\pi} \sum_{|\gamma - 2T| < \delta} f(\gamma - 2T)$$

Zeros near height $2T$ cause fluctuations in $\Theta(T)$.

Example: For $T = 20$, height $2T = 40$, the nearby zeros are $\gamma \approx 30.4, 32.9, 37.6, 40.9, 43.3, ...$

4. Main Result

Theorem (Selberg-Weil Connection):

The scattering phase $\Theta(T)$ for $M = SL(2,\mathbb{Z}) \backslash \mathbb{H}$ connects to Riemann zeros via:

  • $E_\Gamma = -1/4 + O(1/T)$ - analytic, bounded
  • $E_\zeta = O(\log T)$ - arithmetic, from zeros near height $2T$

This parallels how $S(T)$ in Riemann-von Mangoldt fluctuates due to nearby zeros.
