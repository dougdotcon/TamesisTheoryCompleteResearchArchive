4. Class Details

U1/2 — Tamesis Discovery

Formula: $\phi(c) = (1 + c)^{-1/2}$

Origin: Stage 34-37, Tamesis Theory

Examples: Permutation → Random map, Cycle fraction decay

Key: Discrete state space + Bernoulli perturbation + Counting observable

U2 — Lindblad Class

Formula: $\phi(c) = (1 + c)^{-2}$ or $P(s) \sim s^2$

Origin: Lindblad 1976, Dyson 1962

Examples: Quantum dephasing (T2), GUE level repulsion

Key: Continuous Hilbert space + Lindblad dynamics

U0 — Threshold Class

Formula: $\phi(c) \sim \Theta(c_c - c)$

Origin: KAM 1954, Stauffer 1994

Examples: KAM chaos, Percolation, Anderson localization

Key: Sharp phase transition at critical point

5. Key Insight

Universality is About Mechanisms, Not Systems

The same exponent appears in seemingly different systems because they share the same mathematical mechanism:

  • U1/2: Discrete counting with random perturbation
  • U2: Quadratic decay in continuous spaces
  • U0: Threshold crossing with sharp boundary
  • Uexp: Independent Poisson events

The Tamesis discovery (U1/2) is a NEW member of this classification, describing a specific family of discrete-to-random transitions.
