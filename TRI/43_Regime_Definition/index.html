Stage 43: Formal Definition of Regime

Theory of Regime Incompatibility (TRI) | 2026-01-23

1. The Problem

The word "regime" is used constantly in physics:

  • "Quantum regime" vs "classical regime"
  • "Strong field regime" vs "weak field regime"
  • "Relativistic regime" vs "Newtonian regime"

But what IS a regime, mathematically? Until now, it was informal discourse. TRI makes it precise.

2. The Definition

A regime is a triple:

$$R = (S, P, O)$$
Component Meaning Examples
S State Space Type Permutation, Hilbert, Riemannian
P Perturbation Type Bernoulli, Lindblad, Metric
O Observable Type Counting, Spectral, Curvature

4. Regime Invariants

Each regime has structural invariants that do NOT change within the regime:

  • Discrete vs Continuous - state space type
  • Quantum vs Classical - commutativity of observables
  • Measure Type - counting, spectral, or threshold
  • Locality - local vs global perturbations

5. Compatibility Check

Two regimes are compatible iff they share the same invariants:

6. Key Insight

The (S, P, O) signature determines the universality class.

Incompatible invariants ⇒ No unified theory possible.

This is the foundation for TRI's no-go theorems.
