Elastic Memory: The Hysteresis of Spacetime

Douglas H. M. Fulber
Universidade Federal do Rio de Janeiro, Rio de Janeiro, Brazil
(Dated: January 23, 2026)
We propose that spacetime is an elastic medium with memory. Like a polymer, it exhibits hysteresis: the path to equilibrium is different from the path away from it. This "Elastic Memory" explains the arrow of time in gravitational systems and suggests that Gravitational Waves leave a permanent scar on the metric (The Gravitational Memory Effect).

Space forgets slowly. This memory is what we call History.
